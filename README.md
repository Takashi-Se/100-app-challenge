# 100-app-challenge

100アプリ開発チャレンジの記録

## プロジェクト概要
このプロジェクトは、100個のアプリケーションを開発するチャレンジの記録です。

## 開発環境
- Cursor IDE
- Git
- GitHub

## プロジェクト構造
```
100-app-challenge/
├── README.md
├── .gitignore
└── apps/          # 各アプリケーションのディレクトリ
```

## 開発フロー
1. 新しいアプリケーションの開発を開始する際は、`apps`ディレクトリに新しいディレクトリを作成
2. 各アプリケーションの開発状況は、このREADMEに記録
3. コミットメッセージは具体的な内容を記載
